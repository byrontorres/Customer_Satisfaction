Identificacin y Manejo de Valores Nulos

Uno de los primeros pasos en el proceso de limpieza de datos es identificar y manejar los valores nulos. Aqu, se calcula la cantidad total de filas en el DataFrame, se identifican los valores nulos en cada columna y se decide cmo tratarlos segn su proporcin.

Para empezar, se cuenta el nmero total de filas en el DataFrame. Este valor es necesario para calcular el porcentaje de valores nulos en cada columna. Se ha establecido como umbral el 0.5% asi:

  • Eliminar filas con valores nulos si el porcentaje es menor o igual a 0.5%
  • Imputar valores nulos con la mediana de los ltimos 10 registros si el porcentaje es mayor a 0.5%

Eliminacin de Outliers y Registro de Cantidad Eliminada

En este segmento de cdigo, se define una funcin para eliminar outliers de un DataFrame utilizando el rango intercuartil (IQR). Tambin se aplica esta funcin a columnas especficas del DataFrame para limpiar los datos y registrar cuntos outliers fueron eliminados.

Definicin de la Funcin para Eliminar Outliers

Se crea una funcin remove_outliers que toma un DataFrame y el nombre de una columna como entrada, calcula los lmites inferior y superior usando el IQR, y elimina los valores que caen fuera de este rango. La funcin tambin cuenta cuntos outliers se eliminaron.

In[42]:
# Definir una funcin para eliminar outliers y contar cuntos se eliminaron
def remove_outliers(df, column):
    Q1 = df[column].quantile(0.25)
    Q3 = df[column].quantile(0.75)
    IQR = Q3 - Q1
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR
    # Contar los outliers
    outliers = df[(df[column] < lower_bound) | (df[column] > upper_bound)]
    count_outliers = len(outliers)
    # Eliminar los outliers
    df_clean = df[(df[column] >= lower_bound) & (df[column] <= upper_bound)]
    return df_clean, count_outliers

Importancia de la Generacin de Datos Sintticos

SMOTE (Synthetic Minority Over-sampling Technique) es una tcnica de sobremuestreo que ayuda a superar el problema de desbalance de clases generando ejemplos sintticos de la clase minoritaria. Aunque el desbalance inicial no era extremadamente pronunciado, la decisin de utilizar SMOTE se tom para minimizar cualquier riesgo de sesgo en los modelos predictivos que se desarrollarn a partir de estos datos.

Anlisis de Multicolinealidad y Eliminacin de Variables con VIF Alto

La multicolinealidad es un problema comn en los modelos de regresin, donde dos o ms variables predictoras estn altamente correlacionadas, lo que puede afectar la precisin y estabilidad del modelo. Para abordar este problema, calculamos el Factor de Inflacin de la Varianza (VIF) y eliminamos las variables con un VIF alto.

Adicin de una Constante y Clculo del VIF

Se comienza aadiendo una constante al conjunto de datos, lo que es necesario para el clculo correcto del VIF. Luego, se calcula el VIF para cada variable en el DataFrame.
